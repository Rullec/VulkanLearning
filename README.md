# SimpleClothSimulator

It's an experimental cloth simulator. Four integration schemes have been implemented in this repo, including:
1. Semi-implicit
2. Full implicit (Newton iteration)
3. Position based simulation (PBD)
4. Projective dynamic (PD)
![2021-04-13-16-09-28](https://raw.githubusercontent.com/Rullec/Images/main/vscode/2021-04-13-16-09-28.gif)

Both stretch and bending forces are supported in these methods.
